description: Created Git-like version control system that uses custom data structures called “blobs” to track file modifications in a directory over time, allowing users to save snapshots of their files and restore them to any previous state. Used cryptographic hash function to uniquely label blobs and maintained a history of mappings between saves and blobs, simulating a tree structure that enables users to extend and branch their modification history memory-efficiently.
